# Aurum House - Premium Membership & Events Frontend

React, Tailwind CSS, and GSAP implementation for a luxury membership and events website.

## Stack

- React with Vite
- Tailwind CSS v4
- GSAP + ScrollTrigger
- React Router
- Mock CMS/API-ready data in `src/data.ts`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Frontend scope

The site includes public pages for Home, About, Membership, Events, Event Details, News, Member Portal, and Contact. Forms and login are frontend-only and intentionally expose clean integration points for future backend authentication, CMS content, event registration, and membership enquiry APIs.
